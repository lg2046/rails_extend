# RailsExtend

Rails extend

## Installation

Add this line to your application's Gemfile:

    gem 'rails_extend', :git => "git://github.com/lg2046/rails_extend'

And then execute:

    $ bundle or bundle update

Or install it yourself as:

    $ gem install rails_extend
